What does it mean to display inline vs inline blocks?

What is it?

Display inline and inline blocks can be seen as a way in which a box appears and wraps around the screen. This box can include anything from images to text. Say that you are trying to place an image within the bounderies of a text, you would use display:inline or display:inline-block to achieve this.

Here are some examples:

display:inline

When looking at inline, this certain box or element which we would like to place will sit on top of the other element, disregarding any margins or paddings within the existing element.

display:inline-block

When using inline-block it carries a similar role to inline except this element will respect the width, height and padding of the already existing element, but can still sit within that same region.
